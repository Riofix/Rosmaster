## 📋 项目概述

**Rosmaster** 是一个基于 **ROS 2 (Humble)** + **STM32F103C8T6** 的智能机器人控制系统，面向竞赛场景实现自动化物料搬运与分拣任务。

系统分为两大子系统：

| 子系统 | 硬件平台 | 技术栈 | 职责 |
|:---:|:---:|:---|:---|
| **上(1)位机** | RDK X5 / 工控机 | ROS 2 Humble + Python | 视觉识别、任务调度、闭环控制、数据融合 |
| **下位机** | STM32F103C8T6 | C (ARM MDK) | 底盘驱动、步进电机控制、传感器采集、舵机/PWM控制 |

## 🧩 ROS2 模块架构（7层解耦）

系统采用 **7层模块化架构**，每层各司其职、高度解耦：

### 1. 📡 链路层 — `robot_link`
与硬件物理通信的桥梁，包含三个节点：
- **SerialNode** — 通过 USB 串口 (115200, 8N1) 与 STM32 底盘通信，收发 `/dev/rosmaster`
- **TcpServerNode** — TCP 服务器 (端口 8080)，与三个抓手的 ESP8266 WiFi 模块通信
- **HotspotNode** — 自动配置 WiFi 热点，提供局域网通信基础

### 2. 📦 协议层 — `robot_protocol`
- **ProtocolNode** — 解析来自硬件的 `0xFF 0xFB` 协议帧，将二进制流转换为结构化影子状态
- **ProtocolPackNode** — 将上层指令打包为 `0xFF 0xFC` 协议帧，路由分发至目标硬件

### 3. 🔗 融合层 — `robot_fusion`
- **FusionNode** — 聚合所有物理设备的状态和视觉识别结果，维护统一的 **全局影子状态** (World State)，50Hz 高频发布

### 4. 🎮 控制层 — `robot_control`
- **ControlNode** — 指令中台，将大脑的语义指令映射为具体硬件控制信号
- 内置 **工业级 PID 控制器**（含加速度斜坡限制），实现底盘精确位置闭环
- 管理 TaskID 生命周期，确保状态一致性

### 5. 🧠 大脑层 — `robot_brain`
- **BrainNode** — 顶层决策状态机，实现完整的竞赛自动化流程：
  > INIT → WAIT_VISION → MOVE_WORK → GRABBING → MOVE_DROP → UNLOADING → FINISHED

### 6. 👁️ 视觉层 — `robot_vision`
- **VisionNode** — 双目摄像头远距离 (1.5m) 数字序列识别
- 基于 OpenCV 透视变换 + 模板匹配 + 帧融合投票算法
- 支持 Debug 可视化窗口

### 7. 🚀 启动层 — `robot_bringup`
- **hardware_launch.py** — 一键启动所有硬件相关节点

## 🔩 STM32 下位机固件

下位机固件运行于 **STM32F103C8T6** 微控制器，负责底层硬件实时控制。

| 模块 | 功能 |
|:---|:---|
| **APP (应用层)** | 主循环调度、无刷电机控制、IMU 姿态解算、步进电机控制、RGB LED、舵机控制 |
| **BSP (板级支持包)** | ESP8266 WiFi、I2C、MPU6050 (六轴姿态)、PWM、SysTick、TCS3472 (颜色识别)、USART、OLED 显示、EMM V5 步进驱动 |
| **Protocol (协议层)** | 无锁环形队列通信、0xFF 0xFB/0xFC 帧解析、校验与分发 |

详见 [src/Stm32F103C8T6/README.md](./src/Stm32F103C8T6/README.md)

## 🚀 快速开始

### 上位机 (ROS 2)

```bash
# 1. 编译所有包
cd ~/robot_ws
colcon build --symlink-install
source install/setup.bash

# 2. 一键启动所有硬件节点
ros2 launch robot_bringup hardware_launch.py

# 3. 单独启动视觉节点（可选）
ros2 run robot_vision vision_node

# 4. 启动大脑（自动化竞赛流程）
ros2 run robot_brain brain_node
```

### 下位机 (STM32)

1. 使用 **Keil MDK** 打开 `MDK-ARM/` 工程文件
2. 编译并下载至 STM32F103C8T6
3. 配置 ESP8266 WiFi 参数（SSID/密码/服务器IP）

## 📌 通信协议

上下位机之间采用自定义帧协议通信：

```
| 帧头1 | 帧头2 | 长度 | 数据段 | 校验和 |
| 0xFF  | 0xFB  | 1B   | N Byte | 1 Byte |   (上行: 硬件 → 上位机)
| 0xFF  | 0xFC  | 1B   | N Byte | 1 Byte |   (下行: 上位机 → 硬件)
```

- 小端字节序 (Little-Endian)
- 校验和: 从长度字段开始累加至校验前，对 256 取余
